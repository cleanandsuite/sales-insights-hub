# SellSig Progression & Leveling Engine

## How Agents Grow Through the System

---

## The Leveling System

### Overview
Every agent starts at Level 1 and earns XP through activities, achievements, and performance. Levels unlock abilities, perks, and rank eligibility.

---

## Rank Promotions

### Requirements by Rank

#### RECRUIT → AGENT
- **Level:** 10+
- **XP:** 4,000+
- **Certifications:** Recruit Core Training
- **Performance:** 50+ calls, 5+ demos
- **Time in Role:** Min 30 days
- **Approval:** Team lead

#### AGENT → SENIOR AGENT
- **Level:** 20+
- **XP:** 22,000+
- **Certifications:** Agent Core Training, 1 Specialization
- **Performance:** 10+ closed deals, 1+ referral
- **Time in Role:** Min 6 months
- **Approval:** Agency Lead

#### SENIOR AGENT → AGENCY LEAD
- **Level:** 30+
- **XP:** 80,000+
- **Certifications:** Leadership Training
- **Performance:** Top 25% for 3+ months
- **Mentorship:** 5+ promoted agents
- **Time in Role:** Min 1 year
- **Approval:** CEO/Board

## Decay System

### Weekly Decay (Prevents stagnation)
- **Base decay:** -5% of current level XP if no activity for 7 days
- **Cap:** Can't decay below previous level threshold
- **Warning:** Notification at 5 days inactive
- **Grace period:** No decay for first 30 days after promotion

### Protected XP
- Level threshold XP never decays (you keep your level)
- Only "extra" XP above threshold decays
- Special badges grant permanent immunity (Mythic badges)

## Leaderboard Systems

### Daily Leaderboard
- **Metric:** Calls made today
- **Reset:** Midnight
- **Reward:** "Top Caller" badge for 24h

### Weekly Leaderboard
- **Metric:** Total XP earned
- **Reset:** Sunday midnight
- **Reward:** Weekly XP boost winner (+25% next week)

### Monthly Leaderboard
- **Metric:** Revenue generated
- **Reset:** End of month
- **Reward:** Top 3 get monetary bonus ($500/$300/$200)

### All-Time Leaderboard
- **Metric:** Total lifetime XP
- **Never resets**
- **Reward:** Hall of Fame on company website

## Anti-Exploitation Rules

### Prevented Behaviors
- Spamming calls (under 30s, no engagement) = 0 XP
- Gaming the system (fake demos) = account review
- Collusion (fake referrals) = termination
- Stacking multipliers = capped at 2x max

### Monitoring
- AI analyzes call quality for legitimacy
- Peer review for manual XP claims
- Random audits of streaks and achievements
