## Using This System

### Implementation Order
1. Deploy Common/Uncommon badges (easy wins)
2. Add Rare badges for key behaviors
3. Launch Epic/Legendary with special rewards
4. Mythic as "once in lifetime" achievements

### Tracking
- Automatic: Calls, deals, metrics
- Manual: Culture badges (peer nominations)
- Hybrid: Streaks (auto-tracked, manually reviewed)

### Notifications
- **Immediate:** Toast notification on earn
- **Daily:** Badge digest in morning briefing
- **Weekly:** Leaderboard update
- **Monthly:** Badge showcase newsletter
